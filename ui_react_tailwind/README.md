# UI React + Tailwind (RTL)

## Installation
1. `cd ui_react_tailwind`
2. `npm.cmd install`
3. `npm.cmd run dev`

## Fichiers clés
- `src/components/EventEditModal.jsx`: modal RTL (desktop + mobile full screen)
- `src/App.jsx`: exemple d'utilisation
- `src/index.css`: style Tailwind + tokens de base

## Notes
- Le footer est sticky avec 2 actions: `ביטול` et `שמירה`.
- Validation incluse: champs obligatoires + `שעת סיום > שעת התחלה`.
- Suppression d'un événement existant avec confirmation.
